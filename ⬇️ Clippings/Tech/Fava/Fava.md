# Fava Web Interface

Fava is a web interface for Beancount, prioritizing features and usability. It offers an online demo at [https://fava.pythonanywhere.com](https://fava.pythonanywhere.com).

## Navigation

*   **Contents:**  Provides access to the main sections of the documentation.
*   **Menu:**  Offers navigation options, including toggling the navigation sidebar and color theme.
*   **API Documentation:**  Detailed documentation for the Fava API, including modules like `fava`, `fava.beans`, `fava.core`, and `fava.util`.  Includes links to the GitHub repository and a chat channel.
*   **Getting Started:**  Provides initial guidance on using Fava.
*   **Changelog:**  Lists updates and changes to the Fava project.
*   **Development:**  Information for developers contributing to Fava.
*   **Issue Tracker:**  A link to the GitHub issue tracker for reporting bugs and suggesting improvements.

## Features

Fava's core focus is on a user-friendly web interface for Beancount double-entry bookkeeping.  It includes a demo instance for immediate testing.
