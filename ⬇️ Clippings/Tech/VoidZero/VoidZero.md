> [voidzero.dev > void(0)](https://voidzero.dev/)

VoidZero is building the next generation of JavaScript tooling, focusing on productivity for web developers. Key projects include Vite (a build tool), Vitest (a test runner), and Rolldown (a JavaScript bundler). The company emphasizes contributing to critical infrastructure projects within the JavaScript ecosystem. They are running a v1.0 prod server. Their mission is to make web developers more productive than ever before.
