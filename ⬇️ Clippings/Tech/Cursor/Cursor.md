> [cursor.com > Cursor - The AI Code Editor](https://www.cursor.com/en)

Cursor is an AI code editor designed to dramatically increase productivity. It leverages a combination of purpose-built and frontier models to predict your next code edits, answer questions from your codebase, and allow you to write code using natural language prompts. Key features include intelligent code completion, one-click code integration, and the ability to update entire classes or functions with simple prompts. Trusted by companies like Stripe, OpenAI, and Johnson & Johnson, Cursor is built to accelerate software development. It offers a seamless experience for coding with AI, focusing on speed, familiarity, and intelligent assistance. The editor is available for Windows and provides access to a comprehensive suite of features, including Frontier Intelligence and a focus on building software faster.
