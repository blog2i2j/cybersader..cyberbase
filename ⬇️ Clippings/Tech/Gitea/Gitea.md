# Gitea Overview

Gitea is a self-hosted Git service designed for teams and developers, offering a streamlined DevOps experience. It provides a comprehensive suite of tools, including Git hosting, code review, team collaboration, package registries, and CI/CD capabilities.

## Product Offerings

*   **Gitea Cloud:** A cloud-based solution for launching dedicated Gitea instances, suitable for individual developers and enterprise users. It offers the fastest way to deploy a Gitea instance.
*   **Gitea Enterprise:** An enhanced DevOps instance for enterprise users, allowing deployment anywhere and providing an enhanced experience.
*   **Gitea:** A free DevOps instance for self-hosting.
*   **Gitea Actions:** Automates Gitea workflows.
*   **Tea:** A command-line tool for interacting with Gitea servers.

## Key Features

*   **Private, Fast, Reliable DevOps Platform:** Gitea facilitates high-efficiency operations from planning to production.
*   **Cloud-Based Deployment:** Gitea Cloud simplifies deployment and management.
*   **Self-Hosted Option:** Provides full control and customization.
*   **Community Support:** Extensive resources and support available through forums, chatrooms, and open-source contributions.

## Target Users

*   **Individual Developers:** For personal projects and development.
*   **Enterprise Users:** For teams and organizations requiring a robust DevOps platform.
