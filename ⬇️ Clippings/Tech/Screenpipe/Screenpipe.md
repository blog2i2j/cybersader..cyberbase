# Screenpipe AI SDK

Screenpipe is a computer use AI SDK that securely captures and analyzes computer screen and audio activity. It allows users to install hundreds of AI agents to process this data. Key features include:

*   **Secure Screen & Audio Capture:** Records screen and audio activity for analysis.
*   **AI Agent Ecosystem:** Supports installation of hundreds of AI agents.
*   **Data Visualization:** Provides a data table to visualize recorded data.
*   **AI App Store:** Offers a community-built app store with plugins.
*   **Time-Based Analysis (Rewind):** Allows users to scroll back in time and ask AI questions about past screen activity.
*   **Speaker Identification:** Enables the assignment of names to voices for consistent identification across recordings.
*   **Interactive Playground (Example-pipe):** A flexible environment for testing and exploring components, including health status checks, UI recordings, OCR, audio transcription, and screen/audio streaming.
