# Canarytokens Overview

Canarytokens is a free tool designed to detect breaches by enabling attackers to announce their presence. These tokens are deployed within a network, triggering alerts when an attacker interacts with them. The system provides notifications upon detection, facilitating rapid response to security incidents.

## Key Features

*   **Alerting Mechanisms:** Canarytokens offers a variety of alert triggers, including:
    *   URL Visits
    *   DNS Resolution
    *   Credit Card Usage
    *   QR Code Scans
    *   MySQL Dump Loading
    *   AWS API Key Usage
    *   Fake App Installations
    *   Log4shell Vulnerabilities
    *   Fast and Slow Redirects
    *   Sensitive Command Execution
    *   Windows Fake File System Access
    *   Web Image Views

## Deployment

The system requires deployment of the tokens within the target network.  The tokens are designed to be versatile and adaptable to various environments.
