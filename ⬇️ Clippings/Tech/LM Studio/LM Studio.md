## LM Studio: Local LLM Discovery and Execution

LM Studio is a platform for discovering, downloading, and running Large Language Models (LLMs) locally on your computer. It supports models like Llama, Gemma 3, and DeepSeek.

### Key Features

*   **Local Execution:** Run LLMs without relying on cloud services.
*   **Model Catalog:** Access a diverse collection of open-source models.
*   **Easy to Use:** Designed for users with varying levels of technical expertise.
*   **Chat and Server Options:** Utilize LLMs for conversational AI or establish local LLM servers.
*   **Document Chat:** Integrate LLMs with documents for enhanced context.

### Supported Models

*   Llama
*   Gemma 3
*   DeepSeek
*   Qwen
*   Phi
