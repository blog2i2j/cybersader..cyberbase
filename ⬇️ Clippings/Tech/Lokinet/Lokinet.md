# Lokinet: Anonymous Internet Access

Lokinet is an onion-router built on LLARP, the fastest onion-routing protocol, providing anonymous internet access. It operates through a decentralized network of staked nodes, ensuring censorship resistance and preventing surveillance.

## Key Features

*   **Onion-Routed Traffic:** All browsing traffic is securely routed through the onion network, guaranteeing privacy and anonymity.
*   **No IP Addresses:** Lokinet hides your IP address and the IP addresses of the websites you visit, masking your location and identity.
*   **Decentralized Network:** The network is powered by a decentralized network of nodes, making it resistant to shutdowns and surveillance.
*   **Cross-Platform Compatibility:** Clients are available for Windows, Linux, and macOS, with mobile clients currently in development.
*   **Easy Integration:** Lokinet can be used with any application, facilitating seamless integration with existing tools.
