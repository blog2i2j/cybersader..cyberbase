# GitButler

GitButler is a Git client that allows users to manage branches, commit changes, and automate workflows. It provides features like virtual branches, quick commits, and branch management. The user can use it to work with multiple Git repositories seamlessly and fix issues while working on features.

Why use GitButler?

GitButler is compatible with existing workflows, writes plain Git trees, and offers features like undo, squash, and amend. It also has a business model where it's free for individual developers and plans to offer team features for companies. The client is designed to be user-friendly and supports both terminal and future terminal interfaces.

Features:

- Branch Management
- Virtual Branches
- Quick Commits
- Undo, Squash, Amend
- Git Workflow
- FAQ
