# Brie.fi/ng - Secure Video Chat

Briefing is a secure, anonymous video chat application built for privacy. It utilizes WebRTC for direct peer-to-peer connections, eliminating the need for a central server and minimizing data storage. Key features include:

*   **End-to-End Encryption:** Leverages WebRTC for secure communication.
*   **No Server Storage:**  The website does not store conversation content or user accounts.
*   **No Cookies:**  Respects user privacy by avoiding cookies.
*   **Direct Peer-to-Peer:**  Connects participants directly, reducing bandwidth usage.
*   **SFU-Free:**  Avoids reliance on a central SFU (Software Functionality Unit) to maintain end-to-end encryption.

**Technical Details:**

*   **Technology:** WebRTC, Vue.js
*   **Purpose:** Secure group video conferencing
*   **Developer:** Holtwick.de
