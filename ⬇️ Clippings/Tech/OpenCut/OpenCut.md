# OpenCut Video Editor

A simple yet powerful browser-based video editor designed for creators who need to edit videos quickly and efficiently.
## Why?

- **Privacy**: Your videos stay on your device
- **Free features**: Every basic feature of CapCut is paywalled now
- **Simple**: People want editors that are easy to use - CapCut proved that
## Key Features

- Browser-based: Works on any device, no installation required.
- Open Source: Community-driven development with a growing number of features.
- Lightweight: Fast performance without compromising functionality.
- Cross-platform: Edits video files seamlessly across different operating systems.

## Description

OpenCut is an open-source video editor that provides users with a robust set of tools to edit their videos. Its simplicity and powerful functionality make it accessible for both casual users and professional creators.
