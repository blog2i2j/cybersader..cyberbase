> [raycast.com > Raycast - Your shortcut to everything](https://www.raycast.com/)

Raycast is a productivity tool that provides a central launcher for powerful productivity tools. It's available for macOS and has a waitlist for Windows. Version 1.94.0 supports macOS 12+ and can be installed via Homebrew. Key features include an AI extensions feature and support for teams. The interface allows users to filter entries by type (text, images, files, links, colors) and includes a search function. The tool is designed for efficiency and ergonomic use. It offers a single interface for accessing all needed productivity tools. The tool is designed for indexing and efficient use.
