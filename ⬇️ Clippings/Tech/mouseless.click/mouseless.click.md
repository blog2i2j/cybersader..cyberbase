# Mouseless: Keyboard-Based Mouse Control

## Overview

Mouseless is a software application that allows users to control their computer mouse using only the keyboard. It provides lightning-fast mouse control, enabling users to click, move, drag, drop, scroll, and perform various mouse actions with minimal typing. The application is designed for users who require an alternative to traditional mice, particularly those with accessibility needs or those seeking to improve their productivity.

## Key Features

*   **Keyboard-Based Mouse Control:** Enables users to perform all standard mouse actions using keyboard shortcuts.
*   **Rapid Clicking:** Allows users to click anywhere on the screen in less than 2 seconds.
*   **Comprehensive Functionality:** Supports double and triple clicks, left, right, middle, back, and forward buttons.
*   **On-Screen Overlay:** Provides a visual guide to assist users in entering coordinates.
*   **Accessibility:** Designed for users who cannot use mice or who require an efficient alternative.
*   **Productivity Enhancement:** Saves time by eliminating the need for reaching for one-off clicks.
*   **Ergonomic Benefits:** Reduces strain and injury associated with prolonged trackpad use.

## Technical Details

The application is available for macOS, Windows, and Linux. It utilizes a command-line interface for installation and can be downloaded from the official website or GitHub repository. The software requires JavaScript to function properly.
