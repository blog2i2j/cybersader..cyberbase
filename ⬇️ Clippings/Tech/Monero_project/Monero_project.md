# Monero Project Overview

This page serves as the homepage for the Monero Project, a digital currency focused on security, privacy, and untraceability. It provides resources and information for users interested in learning about and using Monero.

## Getting Started

The page offers several pathways for new users:

*   **What is Monero?** - Explains the core concepts and features of Monero.
*   **Accepting** - Instructions on how to accept Monero payments.
*   **Contributing** - Opportunities to contribute to the Monero project.
*   **Mining** - Information on how to mine Monero.
*   **FAQ** - Frequently asked questions about Monero.
*   **Downloads** - Links to download Monero software.
*   **Blog** - Updates and news related to the Monero project.

## Community

The page highlights various community initiatives:

*   **Workgroups** - Collaborative groups focused on specific aspects of Monero development.
*   **Hangouts** - Online community gatherings.
*   **Sponsorships** - Support opportunities for the Monero project.
*   **Merchants & Exchanges** - Resources for businesses accepting Monero.
