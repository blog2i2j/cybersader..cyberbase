# Actual Budget Overview

Actual Budget is a fast, privacy-focused finance management application built upon the Envelope Budgeting methodology. It emphasizes user control and data ownership, offering multi-device sync and optional end-to-end encryption.

## Key Features

*   **Envelope Budgeting:** Utilizes the established Envelope Budgeting system for effective budgeting.
*   **Data Ownership:** Users retain complete control over their data.
*   **Multi-Device Sync:** Seamless synchronization across multiple devices.
*   **End-to-End Encryption:** Provides enhanced data security with optional encryption.
*   **Local-First Design:** Operates primarily offline, syncing data in the background.
*   **Streamlined Interface:** Offers a minimal and intuitive user experience.
*   **Split Transactions:** Supports the management of split transactions and transfers.
*   **Automated Reports:** Includes net worth and cash flow reports.

## Installation & Setup

*   **PikaPods Setup:** Quick setup in 2 minutes via PikaPods.
*   **Manual Setup:** Available documentation for manual installation.
*   **Demo:** A live demo is available for testing.

## Design & Technology

*   **Beautifully Designed Interface:** Focused on speed and efficiency.
*   **Local-First Software:** Prioritizes offline functionality and data privacy.
*   **End-to-End Encryption:** Ensures secure data transmission.

## Budgeting & Transactions

*   **Transaction Management:** Streamlined transaction entry and categorization.
*   **Split Transactions:** Supports complex transaction splitting.
*   **Reporting:** Provides insights into net worth and cash flow.

## Getting Started

Actual Budget is designed to be simple and intuitive.  The app is built around the principles of Envelope Budgeting, empowering users to take control of their finances.
