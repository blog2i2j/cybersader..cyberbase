> [cryptpad.fr > CryptPad: end-to-end encrypted collaboration suite](https://cryptpad.fr/)

CryptPad is an end-to-end encrypted collaboration suite, the flagship instance of which is hosted in France and administered by the CryptPad development team. It offers features such as secure document creation, collaborative editing, and data hosting within France. Key features include: a pricing page, documentation, login, signup, a status page, and the ability to create new documents (pad, sheet, document, presentation). The suite prioritizes data privacy and security through end-to-end encryption. The website provides access to terms of service, a privacy policy, and a contact form.  It emphasizes open-source development and a commitment to data residency.
