# Grayjay App Overview

Grayjay is an application designed to allow users to follow creators directly, rather than relying on centralized platforms. The core principle is to empower users to retain ownership and control over their video content consumption.

## Key Features

*   **Creator Following:** Enables users to directly follow creators.
*   **Platform Independence:**  Allows users to watch content from various sources without being tied to a single platform.
*   **Content Ownership:** Users maintain control over the content they watch.
*   **Source Aggregation:** Users can add sources from different services (e.g., YouTube, Rumble) to create a personalized viewing environment.
*   **Downloadable Apps:** Available for Android (Universal, arm64-v8a, arm64-v7a, x86_64, x86) and desktop.
