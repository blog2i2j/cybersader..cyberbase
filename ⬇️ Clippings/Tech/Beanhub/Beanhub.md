# BeanHub: Smart Accounting Book

BeanHub is a modern accounting book app built on Git and Beancount. It simplifies personal finance, budgeting, and business bookkeeping.

## Key Features

*   **Automation:** Automates routines like importing transactions from banks via import rules and custom forms.
*   **Direct Connect:** Pulls bank transactions as CSV files directly into your local filesystem.
*   **Inbox Feature:**  Allows forwarding invoices and receipts for automatic AI processing.
*   **Financial Insights:** Provides diagrams and tools for analyzing financial data and making informed decisions.

## Core Technology

BeanHub leverages Git for version control and Beancount for its text-based double-entry accounting book software foundation.  It's designed to empower users with streamlined financial management.

## Benefits

*   Simplified Financial Tracking
*   Automated Data Input

*   Enhanced Financial Analysis
