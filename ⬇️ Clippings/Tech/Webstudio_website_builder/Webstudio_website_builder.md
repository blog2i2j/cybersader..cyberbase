> [webstudio.is > Advanced Open Source Website Builder](https://webstudio.is/)

Webstudio is an open-source website builder focused on maintainability and speed, utilizing modern web standards. It addresses limitations of existing platforms by offering a visual development tool that outputs fast sites, with a focus on avoiding hosting lock-in and class rearrangement issues. Key commitments include doing one thing extremely well, aligning with web standards, and leveraging the ecosystem for other tasks. The builder is designed for creators seeking a robust and flexible solution, emphasizing maintainability and performance. It utilizes Airtable and Supabase.
