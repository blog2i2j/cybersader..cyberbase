> [tubearchivist.com > TubeArchivist](https://www.tubearchivist.com/)

TubeArchivist is a self-hosted YouTube media server. It provides a way to download and archive YouTube videos. Key features include a browser extension (Tube Archivist Companion) for Chrome and Firefox, a Docker image, project documentation, and a GitHub repository. The project is supported by a community on Reddit and Discord. The website also includes links to the GitHub repository, Docker Hub, and the documentation. The core functionality is centered around downloading and archiving YouTube videos, offering a self-hosted solution for managing and preserving YouTube content.
