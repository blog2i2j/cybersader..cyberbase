# Invidious Overview

Invidious is an open-source alternative front-end to YouTube.

## Key Features & Information

*   **Open Source:** Developed and maintained by the iv-org community.
*   **Alternative to YouTube:** Provides a decentralized and privacy-focused alternative.
*   **GitHub Repository:** Hosted on GitHub ([https://github.com/iv-org/invidious](https://github.com/iv-org/invidious))
*   **Screenshots:** Available at [https://invidious.io/screenshots/](https://invidious.io/screenshots/)
*   **Documentation:** Comprehensive documentation is available at [https://docs.invidious.io/](https://docs.invidious.io/)
*   **Team:** The project is supported by a dedicated team.
*   **Contribution:**  You can contribute to the project via GitHub: [https://github.com/iv-org/invidious#contribute](https://github.com/iv-org/invidious#contribute)
*   **Donations:** Support the project through donations: [https://invidious.io/donate/](https://invidious.io/donate/)
*   **Contact:**  For inquiries, contact the team at [https://invidious.io/contact/](https://invidious.io/contact/)
*   **Licensing:** Licensed under the AGPL-3.0 license: [https://www.gnu.org/licenses/agpl-3.0.en.html](https://www.gnu.org/licenses/agpl-3.0.en.html)
*   **Community Chat:**  Engage with the community via Matrix ([https://matrix.to/#/#invidious:matrix.org](https://matrix.to/#/#invidious:matrix.org)), Libera.chat ([https://web.libera.chat/?channel=#invidious](https://web.libera.chat/?channel=#invidious)), and Fediverse (@invidious@social.tchncs.de).

## Usage

*   **Use Invidious:**  Access Invidious instances at [https://instances.invidious.io/](https://instances.invidious.io/)
*   **Host Invidious:**  Self-host the application at [https://docs.invidious.io/installation/](https://docs.invidious.io/installation/)
