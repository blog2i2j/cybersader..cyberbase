# Org Mode

Org Mode is a GNU Emacs major mode for note-taking, document authoring, computational notebooks, literate programming, to-do lists, project planning, and more. It's a versatile plain text system.

## Key Features

*   GNU Emacs Major Mode
*   Note-taking and Document Authoring
*   Computational Notebooks
*   Literate Programming
*   To-Do Lists
*   Project Planning
*   Plain Text System
*   Installation via GNU ELPA
*   Community-Driven Documentation (Worg)
*   Bug Reporting
