# OrbStack: Fast, Light, Simple Docker & Linux

OrbStack is a fast, light, and easy way to run Docker containers and Linux. It offers a drop-in replacement for Docker Desktop, enabling rapid development with Linux machines in just one minute. Key features include:

- **Speed:** Starts in seconds with turbocharged networking, smooth Rosetta x86 emulation, VirtioFS file sharing, and other optimizations.
- **Lightweight:** Low CPU and disk usage, minimal memory consumption, and a native Swift app, eliminating battery drain.
- **Effortless Integration:** Seamless containers, CLI integration, file sharing, and remote SSH editing with Linux machines.
- **Robust Connectivity:** Connect between containers and machines, and use IPv6 painlessly.
- **Drop-in Replacement:** Easily get started with Linux machines in 1 minute with the simple, yet powerful, app and command line.
- **Endlessly Capable:** Run containers, Kubernetes, and Linux distros all with robust integration. Manage containers & machines from anywhere with the menu bar app.

OrbStack is designed to provide a superior development experience by eliminating the complexities and limitations of traditional container solutions. It’s a powerful tool for developers seeking speed, efficiency, and ease of use.
