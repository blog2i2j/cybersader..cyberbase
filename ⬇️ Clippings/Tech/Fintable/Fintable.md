# Fintable Overview

Fintable is a service that synchronizes bank account data to Google Sheets or Airtable. It integrates with over 15,000 banks across 30+ countries. The primary function is to provide users with automated financial data import and analysis capabilities.

## Key Features

*   **Bank Account Sync:** Automatically imports transactions from supported banks.
*   **Google Sheets & Airtable Integration:** Seamlessly connects with popular spreadsheet and database platforms.
*   **Global Coverage:** Supports a wide range of banks worldwide.
