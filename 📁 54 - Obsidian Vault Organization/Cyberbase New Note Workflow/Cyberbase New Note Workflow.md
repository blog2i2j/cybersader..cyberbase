## QuickAdd Overview

> [!summary]
> Quickadd is a combination of 4 tools also known as choices...

### 1) Templates

> [!tldr] add new notes

#### Uses:

- Create new notes
- Use predefined templates
- Keep folders organized

#### Works with:

- Obsidian Templates
- Templater plugin

### 2) Capture

> [!tldr] add new content to certain notes

#### Uses:

- To files and folders
- Insert format syntax
- Insert before/after

#### Works with:

- Active notes
- New notes

### 3) Macro

> [!tldr] advanced automation

#### Uses:

- Powerful tools
- Sequence of commands

#### Works with:

- Obsidian commands
- Editor commands
- User JS scripts
- Choices

### + Multi

> [!summary] Folder to organize Templates, Captures, and Macros
